# Graduate Admission Prediction – Model Accuracy Improvement

## Project Overview
This project focuses on improving the prediction accuracy of graduate admission chances using advanced machine learning techniques.

The project enhances an existing baseline model by applying feature engineering and optimized machine learning algorithms.

## Objective
To improve the predictive performance of the admission prediction model compared to the baseline model.

## Dataset
Graduate Admission Dataset

Key Features:
- GRE Score
- TOEFL Score
- University Rating
- SOP
- LOR
- CGPA
- Research Experience

Target Variable:
Chance of Admission

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Project Workflow

### 1. Baseline Model
Started with the original model used in the class project.

Baseline metrics were evaluated using:
- RMSE
- MAPE
- R² Score

### 2. Model Enhancement
Improved the model using:

- Feature Engineering
- Feature Selection
- Hyperparameter Tuning
- Advanced Regression Models

### 3. Model Comparison
Compared baseline model with improved models.

Performance metrics used:
- RMSE
- MAPE
- R² Score

### Results
The improved model achieved significantly better prediction accuracy compared to the baseline model.

### Insights
Important factors influencing admission chances include:
- CGPA
- GRE Score
- Research Experience
- University Rating

## Practical Application
Universities can use this model to:
- Predict applicant admission probability
- Improve decision-making in admission processes
